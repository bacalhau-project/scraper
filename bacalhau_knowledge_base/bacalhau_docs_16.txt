node.PrerequisitesBacalhau makes the assumption that you have installed all the necessary drivers and tools on your node host and have appropriately configured them for use by Docker.In general for GPUs from any vendor, the Bacalhau client requires:DockerPermission to access DockerNvidiaNVIDIA GPU DriversNVIDIA Container Toolkit (nvidia-docker2)Verify installation byRunning a Sample GPU driversxpu-smitool installed and functionalSee theRunning on GPU under dockerfor guidance on how to run Docker workloads on Intel GPU.GPU Node ConfigurationAccess to GPUs can be controlled
